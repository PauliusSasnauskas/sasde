# Symbolic Neural Architecture Search for Differential Equations

`eq1.ipynb`, `eq2.ipynb`, `eq3.ipynb` contain experiments produced for the paper.

## Requirements
```
sympy >= 1.12
matplotlib >= 3
jax >= 0.4.9
jaxlib >= 0.4.9
jaxtyping >= 0.2.16
optax >= 0.1.5
```

Run `pip install -r requirements.txt` to install.
